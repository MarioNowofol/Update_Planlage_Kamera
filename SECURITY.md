# Sicherheit

Dieses öffentliche Repository enthält keine vollständige Entwicklungsumgebung und keinen Quellcode. Es dient nur zur Bereitstellung von Release-Dateien für automatische Updates.

## Grundprinzip

Das private Entwicklungsrepository `MarioNowofol/Planlage_Kamera` bleibt geschützt. Dieses öffentliche Repository enthält nur die für Ziel-PCs benötigten Update-Dateien.

## Keine Zugangsdaten

In diesem Repository dürfen keine Passwörter, Tokens, internen Zugangsdaten oder vertraulichen Konfigurationsdateien abgelegt werden.

## Release-Dateien

Vor Veröffentlichung eines neuen Releases sollten die Dateien geprüft werden:

- korrekte Versionsnummer
- korrekter Dateiname `Planlage_Kamera.exe`
- passende technische Dokumentation
- nachvollziehbare Patchnotes
